# 🌊 Spiritual Voyage Design: 항해의 영적 치유 여정

> **"우려내는 데는 시간이 필요하다.  
> 빚는 데는 어둠이 필요하다."**
> 
> ---
> **"공간이 영혼을 치유하는 항해가 되다"**
> 
> 본 프로젝트는 항해의 은유를 통해 인간 내면의 깊은 치유와 변화를 안내하는 **융합 예술 공간 기획**입니다.

---

## 🧭 프로젝트 철학 (Philosophy)
영적 치유의 안내자(Shaman)로서, 방문객이 일상을 벗어나 자신의 내면으로 항해할 수 있는 '치유적 시공간'을 설계합니다. 

* **치유와 변화:** 공간에서의 경험이 삶의 변화로 이어지는 트리거가 됩니다.
* **항해의 은유:** 미지의 바다를 항해하듯, 공간의 요소를 통해 자아를 탐험하도록 돕습니다.
